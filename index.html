Bluetooth Connection

KeyFlow uses the Web Bluetooth API (Chrome / Edge only). After pairing, typed characters are forwarded to the device via BLE.

Best results: A BLE HID peripheral (e.g. an ESP32 running BLE HID firmware) acting as a keyboard bridge gives full keystroke injection to any paired host.

No BLE device? Use Copy to paste into any app on any device.
