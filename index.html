Bluetooth Keyboard

KeyFlow uses the Web Bluetooth API to connect to Bluetooth devices. Once connected, typed text is sent to the paired device as keyboard HID reports.

Requirements: Chrome or Edge browser, a Bluetooth HID-capable device, and permissions granted when prompted.

Note: Web Bluetooth HID output is experimental. For full cross-device typing, pairing KeyFlow's output via a dedicated Bluetooth HID adapter (like a microcontroller running BLE HID) is the most reliable approach.

Alternatively, use the Copy button and paste into any app on any device.
